Basic Computer Skills

This module introduces learners to the foundational knowledge and skills required to operate a computer with confidence.

Topics covered include:

  • Understanding computer parts: monitor, CPU, mouse, and keyboard
  • How to use a mouse and keyboard effectively
  • Working with basic applications: Notepad, WordPad, and Paint
  • File management: creating, saving, and organizing folders
  • Introduction to typing and basic shortcuts

With these essential skills, learners can progress to more advanced tasks like using the internet, word processing, and participating in online learning and business activities.
